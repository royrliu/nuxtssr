# nuxt + golang beego api

> Nuxt.js Server Rendered (Universal SSR).  golang beego api project.

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm start
```
